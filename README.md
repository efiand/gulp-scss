# Шаблон для сборки статических страниц c SCSS


## Структура проекта

* `public` - корень сайта.
* `source` - каталог исходников.


## Установка

* Установите Node,js.
* Выполните команду `npm ci`.


## Команды

* `npm run build` - сборка проекта.
* `npm run dev` - сборка проекта и запуск в режиме раработки.
